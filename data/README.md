# data/

No datasets are redistributed in this repository.

The P3HT-CNT dataset used for the retrospective benchmark in Section 2.2 of
the paper is the one released by Bash et al. (*Adv. Funct. Mater.* 2021,
**31**, 2102606), available at:

> https://github.com/Lightmann/GraphModel_for_CNTDesign

The file used by the benchmark is `data_all_back_A.xlsx`. Place it next to
`benchmarks/p3ht_cnt/run_benchmark.py` before running that benchmark; see
`benchmarks/p3ht_cnt/README.md` for details.

The synthetic benchmarks in `benchmarks/synthetic/` do not require any
external data — the three test functions are defined in code.
